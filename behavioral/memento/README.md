# 备忘录模式 (Memento Pattern)

## 概述

备忘录模式用于保存对象的历史状态，并在需要时恢复，而不暴露对象的内部实现细节。

## Go 语言实现特点

- 原发器负责创建和恢复快照
- 管理者负责保存历史记录
- 快照对象通常是一个小型结构体

## 代码示例

当前实现使用一个简单编辑器演示：

- `Editor`：原发器
- `Snapshot`：备忘录
- `History`：管理者

通过 `Save()` 保存快照，`Restore()` 恢复内容。

## 使用场景

1. 编辑器撤销
2. 配置回滚
3. 游戏存档
